Summary of Module Fit Analyses

Fit rules: For dimension D and module M, k = round(D/M). Full-fit if |D − k·M| ≤ tolerance. Half-fit (score 0.5) when not full-fit and |D − k₂·(M/2)| ≤ tolerance for odd k₂. Full-fit: |D − k·M| ≤ tolerance

Shifts: To nearest full multiples down/up
